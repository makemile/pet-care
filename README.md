## Installation
   **1. Clone the repo**

    1. git clone https://github.com/makemile/pet-care

   **2. Navega al directorio del proyecto**

    2. cd pet-care
  **3. Install NPM packages**

    3. npm install shopping-fashion
  **4. Inicia el servidor**

    4. npm run dev
